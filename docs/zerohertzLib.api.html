zerohertzLib.api.api_key(api_key: str) → OpenAIObject¶

위와 같이 OpenAI 페이지에서 발급 후 API를 등록해야 사용할 수 있다.

매개변수:

api_key (str) – OpenAI의 API Key

반환:

사용 가능한 model들의 정보

반환 형식:

openai.openai_object.OpenAIObject

예제

>>> zz.api.api_key("sk-...")

    "data": [

        "id": "text-search-babbage-doc-001",

        },
