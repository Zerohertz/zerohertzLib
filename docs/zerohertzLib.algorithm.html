FFT (Fast Fourier Transform)를 수행하기 위한 함수

Parameters:

  • sig (List[complex]) – 입력 신호 (복소수 리스트)

  • inv (int) – 변환 방향을 지정 (0: 정방향, 1: 역방향)

Returns:

Return type:

List[complex]

Examples

>>> zz.algorithm.fft([1, 0, 0, 0], 0)
[(1+0j), (1+0j), (1+0j), (1+0j)]

[(4+0j), 0j, 0j, 0j]
