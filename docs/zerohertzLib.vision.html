Module contents¶

zerohertzLib.vision.img2gif(images_path: str, output_filename: str | None = 'tmp', duration: int | None = 500) → None¶

Directory 내 이미지들을 GIF로 변환

매개변수:

  • images_path (str) – GIF로 변환할 이미지들이 존재하는 경로

  • output_filename (Optional[str]) – 출력될 GIF 파일명

  • duration – (Optional[int]): ms 단위의 사진 간 간격

반환:

현재 directory에 바로 GIF 저장

반환 형식:

None

예제

>>> import zerohertzLib as zz
>>> zz.vision.img2gif("./")
