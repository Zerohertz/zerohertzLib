### Features

+ `plot`
   + Scatter chart 시각화 함수 `scatter` 추가
   + Pie chart의 label에 소수 자리가 생기는 문제 해결 (resolves: [#40](https://github.com/Zerohertz/zerohertzLib/issues/40))
+ `vision`
  + `cv2` 의존성이 설치되지 않은 경우에도 사용할 수 있도록 변경
  + 이미지의 전후 비교를 위한 `before_after` 함수 개발
  + 여러 이미지를 한번에 요약해 볼 수 있는 `grid` 함수 개발 (resolves: [#39](https://github.com/Zerohertz/zerohertzLib/issues/39))

### Chore

+ Issue 및 PR의 태그 생성 추가 (resolves: [#41](https://github.com/Zerohertz/zerohertzLib/issues/41))
